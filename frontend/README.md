# Sound Sentinel

## Project Overview
Sound Sentinel is an AI-powered noise classification application.

## Getting Started

To run the project locally:

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
    # or
   bun install
   ```
3. Run the development server:
   ```bash
   npm run dev
    # or
   bun dev
   ```

## Technologies
- React
- Vite
- Tailwind CSS
- shadcn/ui
